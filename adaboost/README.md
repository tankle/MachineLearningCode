##Adaboost

Adaboost 的算法过程和推导可以参考《统计学习方法》。


###若分类器算法
...c++
将最小错误率minError设为+∞
对数据集中的每一个特征：
    对每个步长：
        对每个不等号：
            建立一个单层决策树并利用加权数据集对它进行测试
            如果错误率低于minError，则将当前单层决策树设为最佳单层决策树
返回最佳单层决策树
...            


###Adaboost算法
...C++
对每次迭代：
    利用buildStump（）函数找到最佳的单层决策树
    将最佳的单层决策树加入到单层决策树数组
    计算alpha
    计算新的权重向量D
    更新累计类别估计值
    如果错误率等于0.0， 则退出循环
    
...
